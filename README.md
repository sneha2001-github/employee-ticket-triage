# employee-ticket-triage

Role-based employee ticket triage system with AI assist
## Features
- Employee & Admin role-based authentication
- Ticket creation, update, delete (status-based)
- Admin ticket assignment & status management
- Client-side filtering (priority, status, category)
- AI-assisted ticket analysis with fallback handling

## Tech Stack
Frontend: React, Vite, MUI, Redux  
Backend: Node.js, Express, MongoDB  
Auth: JWT, Role-based access  
AI: OpenAI (fallback-ready) / Local LLM (planned)
